# Hangman
Hangman project Lw2
Hangm.py – головний модуль , логіка програми
Модуль StartMenu – Відповідає за стартове меню при запуску програми
Модуль topic – відповідає за базу слів та вибір теми гри
Модуль play_loop – при закінченні гри питає про продовження чи вихід 
Модуль plus – відповідає за відображення картинок при невдалих спробах та інформує те скільки їх залишилось
